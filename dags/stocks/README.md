# MOEX DAG

## Назначение

DAG загружает данные об акциях из MOEX в PostgreSQL

---

## Структура DAG

DAG есть следующие таски:
- ***start*** - EmptyOperator для начала пайплайна
- ***stock*** - PythonOperator для загрузки данных из MOEX
- ***end*** - EmptyOperator для окончания пайплайна

---

## Особенности 

- MOEX не дает выгружать данные за 1 день, так что в api.py выгружается 
2 дня текущий и предыдущий, далее предыдущий удаляется

---

## Настройка конфига

- Конфиг находиться в папке *dags\stocks\config.py*
- Для его настройки необходимо добавить тикер фондовой биржи в TICKERS, например "TSLA"
